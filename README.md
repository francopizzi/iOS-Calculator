# iOS-Calculator

**My first app developed in Swift. It is a calculator, similar to native iOS calculator**

*What mathematical operations can you do?*

- Addition.
- Subtraction.
- Multiplication.
- Division.
- Percent.
- Opposite.
- All this opperations with integers and doubles.
